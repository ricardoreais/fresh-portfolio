# FreshPortfolio

Personal portfolio built with Angular 13, featuring a clean timeline layout to showcase professional experience, projects, and skills.

## Development

```bash
npm install
ng serve
```

Navigate to `http://localhost:4200/`. The app will automatically reload on file changes.

## Build

```bash
npm run build
```

Build artifacts go to `dist/fresh-portfolio`. Uses `--base-href /` for root-level GitHub Pages hosting.

## Deploy

Merges to `master` automatically deploy via [GitHub Actions](.github/workflows/deploy.yml).
